## Setup
Create .env file and add following variable in it:

```
PORT=<PORT>
MONGOSERVER=<MONGOSERVER IP>
DATABASE=<my_database>
```

To run this project, install it locally using npm:

```
$ cd /url_shortner/apis
$ npm install
$ npm run build
$ npm start
$ npm run test

```

Find Documentation on 
``` 
<URl>/api-docs
```
